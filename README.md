# blog

> 一个简单的博客，正在开发中...


## 开发

```bash
npm install
npm run server
```

## Npm 脚本

-   使用 `npm run new <title>` 新建一篇文章
-   使用 `npm run server` 本地预览
-   使用 `npm run publish` 生成并发部文章
-   使用 `npm run clean` 在修改主题无效时清除缓存
